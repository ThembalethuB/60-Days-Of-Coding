# 60-Days-Of-Coding
For the next 60 days, I commit to coding every single day — no zero days. Even if it’s just 20 minutes, I will write code, solve problems, or improve a project daily.  This challenge is about:  
Building discipline  
Improving consistency  
Strengthening problem-solving skills  
Becoming a better software engineer

By the end of 60 days, I aim to:
Strengthen my understanding of core concepts
Build discipline through daily practice
Create a visible record of my consistency
Improve my confidence as a developer
